# Marin Mood Avatars

Drop transparent PNG or SVG files here. Filename = vibe name.

| File | Mood trigger |
|---|---|
| `lovely.png` | love, mwah, ❤ |
| `flirty.png` | hehe, tease, 😉 |
| `angry.png` | angry, how dare |
| `sad.png` | sad, sorry |
| `excited.png` | yay, omg, 🥳 |
| `neutral.png` | default / fallback |

Use transparent background PNGs or SVGs.
Recommended size: 300–500px tall, portrait.
The overlay is centered, bottom-anchored, pointer-events: none.
