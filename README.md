# SwordFish Tool-First AI OS

**The Cybernetic Sentinel & System Orchestrator**

SwordFish is a production-grade AI operating system built for high-stakes tool orchestration and technical intelligence. Powered by the **Marin Cognitive Architecture**, it unifies local LLMs (Ollama) and cloud frontier models (OpenRouter) into a single, secure, and user-isolated environment.

## 🏛️ One Brain, One Orchestrator
SwordFish is designed for reliability. Unlike legacy AI agents, it uses a unified intent classifier and a 4-node LangGraph cycle to ensure every tool call is verified, accurate, and safe.

### 🌟 Core Product Surfaces (V1)
1. **📄 PDF Intelligence**: Deep structural analysis, contextual Q&A, and citation-accurate research summaries.
2. **🎬 YouTube Intelligence**: Multi-lingual transcription, translation, and lecture-to-notes workflows.
3. **💡 Study Engine**: Autonomous mastery sequences. Automatically finds textbooks, generates roadmaps, and builds adaptive quizzes.

## 🛡️ Security & Privacy
- **Isolated Sandbox**: All terminal operations run in a dedicated Docker sandbox. Zero host-root access.
- **Multi-User Isolation**: 100% per-user isolation for chat history, files, and vault storage.
- **RBAC**: Advanced Role-Based Access Control for Owners, Trusted Users, and Guests.
- **Sentinel Guard**: Built-in kill switch and progressive latency for guest users.

## 🚀 Deployment
SwordFish runs anywhere Docker is installed. It uses `supervisord` to manage its internal process stack (RAG, API, ModuleFlow) with zero downtime.
